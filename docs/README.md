- 「関わったサービスの一番のオタクになる」がモットーです
- ビジネスロジックが複雑、かつ、売上に貢献しているシステムを改善するのが好きです

- 美しいシステムよりも、ユーザーに対して早く価値提供できることに価値を感じます
- サーバーサイドの開発が比較的得意ですが、インフラやフロントエンドの開発もできるようになりたいです
